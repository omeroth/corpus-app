# d/

Generated by `npm run gen:share`. One HTML landing page per dialogue ×
language: `d/<weekId>-<dayId>-<lang>.html`.

Each page carries full OG + Twitter card metadata pointing at the
matching PNG in `../share/cards/<lang>/`, plus a minimal styled body
with the question, thinker portrait, era, teaser, and store buttons.

Shared to social platforms as `https://corpusapp.io/d/…`. The app's
completion-modal share flow builds this URL and hands it to the OS
share sheet; the receiving app resolves the OG tags into an inline
preview card.
